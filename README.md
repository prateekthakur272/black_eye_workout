# BlackEye

Track gym workouts and exercise easily on your phone with black eye workout tracker

## Getting Started
1. Clone project repository
```shell
git clone https://github.com/prateekthakur272/black_eye_workout
```

2. Install dependencies
```shell
flutter pub get
```

3. Run Project
```shell
flutter run
```
